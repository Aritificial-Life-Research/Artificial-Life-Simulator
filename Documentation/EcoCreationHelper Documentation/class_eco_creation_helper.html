Give creature the capacity to store a particular resource.

Parameters
resourceEditorWrapper for helping to create a resource.
nameName of resource. Must match name passed into EcoCreator.addResource().
maxLevelMaximum amount that can be stored.
initialLevelInitial amount stored.
healthGainThe amount of health gained when the resource amount surpases gainThreshold.
gainThresholdThe threshold above which healthGain is added to the creature's health each turn.
healthDrainThe amount of health that is lost each turn when the resource amount fall below drainThreshold.
drainThresholdThe threshold below which healthDrain is subtracted from the creature's health each turn.
baseUsageThe baseline amount of the resource that is used every turn.

Set basic parameters for a creature type.

Parameters
ceCreature wrapper.
nameName of creature.
phenotypePhenotype of creature (potentially sensed by other creatures).
turnTimeThe amount of time given for a creature's turn. It's recommended to use the same turn time for all creatures.
maxHealthThe maximum possible health a creature can achieve.
initialHealthThe initial health of the creature when a population is created. Note that a child inherits the health of it's parent, and doesn't use this value.
actionClearIntervalThe action queue is cleared at intervals of this number of steps.
actionClearSizeThe size that the action queue must reach before it is cleared.
mutationDeviationThe initial amount of mutation that occurs with reproduction.
colorThe color of the creature.
usePhenoNetChose wether creature should sense the phenotypes of it's neighbors. If so, you must create a phenotype network template.
mutationDeviationFractionThe number that the mutation deviation is multiplied by each time a creature reproduces. This new deviation is passed on to the child. This can be used to slowy reduce the amount of variability in the population.
lowestMutationDeviationThe lowest value that the mutation deviation can reach, even with the mutationDeviationFraction.
mutationTypeThe type of mutation being used.
